彦达的Ubuntu主题及略

2015.08.03 Yanda Huang Sharing  热度 ℃

Updated on 2016-08-02

Since the latest Ubuntu LTS version is 16.04 but not 14.04 anymore, here’s some new things:

Ubuntu

  • GTK Theme: Arc Theme, note that in Unity environment, there’s no transparent effect
  • Icon Pack: Arc Icon Theme with application icon fallback on Paper Icon Theme

Sublime

Who use Sublime any more! It does not support touch screen!

The most important feature of a text editor must be touch screen support.

Come download the fantastic Visual Studio Code! IT SUPPORT TOUCH SCREEN!

Ubuntu

  • GTK主题: Paper,有ppa源,建议用apt-get install安装,进行跟进同步
  • 图标包: Numix Circle,需要翻墙
  • 主题工具: Unity Tweak Tool sudo apt-get install unity-tweak-tool

Sublime

  • 主题: Material Theme

插件们

  • .h和.cpp之间无缝互相提示
  • 在项目下开terminal
  • 自动生成注释
  • 代码风格优化
  • 免编译校验越界或者语法错误 注意使用这个还要安装cpplinter或者cppcheck,具体略
